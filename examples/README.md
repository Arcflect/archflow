# Examples

This directory contains example Archflow inputs and expected outputs.

Each example is structured as:

- `archflow/`: input files for Archflow
- `expected/`: example output structure after planning or scaffolding

These examples are designed to show how Archflow turns architecture into:

- placement rules
- artifact contracts
- AI handoff prompts
- implementation scaffolding

## Included examples

### `minimal`
The smallest possible example.
Useful for understanding the basic flow.

### `rust-clean-hexagonal`
A Rust-oriented example using a clean / hexagonal style layout.

### `generic-layered`
A language-agnostic layered example for broader adoption.
